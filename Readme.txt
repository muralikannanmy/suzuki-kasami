Suzuki Kasami Broadcast Algorithm implemented in Javascript (NodeJs Runtime)

Platform Dependencies:
----------------------
Operating System: Linux
NodeJS version : 18
(Only Nodejs core modules are used so no additional dependencies are required)

How to run:
-----------
1.run command "node assignment.js" to run the program
2.Enter the no of sites when prompted
3.The program will start and will print the pids of the sites which are created
4.Wait for a few seconds, the algorithm will start running
5.To terminate Ctrl+C can be used
6.If left running the program will self terminate in 1 minute.
